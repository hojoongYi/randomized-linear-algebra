# Randomized Linear Algebra
This repository contains implementations and experiments based on TU Berlin's *Randomized Linear Algebra* course taught by Dr. Oleh Melnyk during the Summer Semester 2026. We study how randomization and techniques from high-dimensional probability theory can be used to improve performance at the cost of reduced accuracy, thereby enabling big data processing.

## Topics covered
- Exercise 01: Randomized Matrix Multiplication


## Structure
Each folder corresponds to one exercise and includes:
- implementation (NumPy, Matplotlib, Scipy, scikit-learn)
- experiments and evaluations
- results and observations
